# backend

## Project setup
> [!WARNING]
> This project requires Python 3.10.0 or higher.

To install the dependencies, run:
```bash
pip install -r requirements.txt
```

## Running the project
To start the project, run:
```bash
python bookquest/app/__init__.py
```

## Linting
To lint the project, run:
```bash
flake8 bookquest
```
